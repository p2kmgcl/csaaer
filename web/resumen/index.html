Introducción

Las funciones resumen o funciones hash son algoritmos de encriptación que, como su nombre indican, transforman un mensaje en una cadena normalmente más pequeña y condensada, denominada "valor hash".

Algunas de las características principales de las Funciones Resumen son:

  • Los hash generados por un mismo método tienen la misma longitud, fuera cual fuese el mensaje claro.
  • Cada hash es único: un texto claro produce un hash propio para el mismo.
  • No se puede recuperar el mensaje original de un valor hash.

    • Las funciones resumen tienen multitud de usos en distintos ámbitos como la integridad informacional, la seguridad (huellas digitales, generación de claves, algoritmos...) o la programación (tablas hash).
